# 🛒 E-Commerce Web Application

This is a full-stack E-Commerce application built using the **MERN stack** (MongoDB, Express.js, React, and Node.js).

## 🚀 Features

- User registration & login
- Product listings
- Add to cart functionality
- Online payments with Stripe
- Order tracking
- Admin dashboard to manage products & orders
- Responsive design

## 🛠️ Tech Stack

- **Frontend**: React, Tailwind CSS, Redux
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT
- **Payment**: Stripe
- **Deployment**: Vercel (Frontend), Render / Railway (Backend)

## 📦 Installation

1. **Clone the repo**

```bash
git clone https://github.com/yourusername/ecommerce-app.git
cd ecommerce-app
```

2. **Setup frontend & backend**

```bash
# For frontend
cd client
npm install
npm start

# For backend
cd server
npm install
npm run dev
```

3. **Environment Variables**

Create `.env` files in the frontend and backend folders with necessary keys like:
- MongoDB URI
- JWT secret
- Stripe keys
- API base URL
